# 南京大学周仁来团队心理学科普实验体验平台

仓库名：`nju-rlz-psych-demos`

本项目用于南京大学周仁来团队开展心理学科普活动，整合适合小学高年级学生体验的互动心理学范式。

## 当前结构

```text

├─ assets/

├─ go-nogo/
│  └─ index.html
├─ visual-illusion/
│  └─ index.html
└─ README.md
```

## 两个区域

- 科技体验区：脑电、心电、VR 眼镜
- 认知挑战区：红蓝机器人、Go/No-Go、视觉错觉、舒尔特、视觉搜索、Stroop、非注意盲视、汉诺塔、传话游戏等
